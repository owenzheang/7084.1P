Event Management App

This is an Android application developed for SIT708 Task 4.1P.

Features

Create new events
View all events in a list (RecyclerView)
Update existing events
Delete events
Input validation (title, date, time)
Date and Time picker
Data persistence using Room database
Bottom Navigation with fragments
  

Technologies used
  
Java
Android Studio
RecyclerView
Fragment & Navigation Component
Room Database (SQLite)

How to Use

Open the app
Navigate to Add Event
Enter:
   Event title
   Category
   Location
   Date and Time
Click <Save> to create a new event
Go to <Event List> to view all events
Use:
   <Edit> to update event
   <Delete> to remove event


Data Storage

This app uses <Room Database> to store data locally on the device.  
All events remain saved even after the app is closed or restarted.
